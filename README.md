# Car-sales-analysis
Sales analysis of the BMW automobile dataset. It was found the best seller models cars by region 

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## How to Run

1. Clone the repository
2. Install dependencies using environment.yml
3. Open the notebook and run all cells
